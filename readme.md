# 兒童遊戲學習 — 遊戲說明

## 🔢 數字記憶配對

**適合年齡：** 4-8 歲  
**遊戲人數：** 1 人  
**訓練能力：** 記憶力、專注力

### 玩法說明

1. 畫面上有 12 張卡片（6 對數字），全部背面朝上
2. 每次翻開兩張卡片，找出相同的數字配對
3. 配對成功後卡片會翻開保留，並有音效與紙片動畫鼓勵
4. 配對失敗卡片會翻回背面，需記住位置再嘗試
5. 成功配對全部 6 對即挑戰成功！

### 操作方式

- **滑鼠點擊**或**觸控螢幕**翻開卡片
- 左上角「🏠 回首頁」可隨時返回首頁
- 遊戲結束後點「🔄 再玩一次」重新開始

## 🎨 顏色大冒險

**適合年齡：** 3-6 歲  
**遊戲人數：** 1 人  
**訓練能力：** 顏色辨識、觀察力

### 玩法說明

1. 畫面中央會出現一個彩色圓形
2. 從下方 4 個顏色方塊中，點選與圓形相同的顏色
3. 答對會有音效與紙片動畫慶祝
4. 答錯會顯示正確答案，幫助小朋友學習
5. 共 8 題，答對越多得分越高！

### 操作方式

- **點擊顏色方塊**作答
- 左上角「🏠 回首頁」可隨時返回首頁
- 答完所有題目後點「🔄 再玩一次」重新開始

### 出現的顏色

紅色、藍色、黃色、綠色、紫色、橘色、粉紅色、咖啡色、黑色、灰色

## 🐾 動物認知

**適合年齡：** 3-7 歲  
**遊戲人數：** 1 人  
**訓練能力：** 動物認知、記憶力

### 玩法說明

1. 畫面上方會出現一個動物圖案與牠的叫聲提示
2. 從下方 4 個選項中，選出正確的動物名稱
3. 答對會有音效與紙片動畫鼓勵
4. 答錯會顯示正確答案，加深印象
5. 共 10 題，答對 8 題以上獲得「太厲害了」評價！

### 操作方式

- **點擊按鈕**作答
- 左上角「🏠 回首頁」可隨時返回首頁
- 答完所有題目後點「🔄 再玩一次」重新開始

### 認識的動物

小狗、小貓、兔子、乳牛、小豬、母雞、綿羊、小馬、大象、獅子、老虎、熊熊、熊貓、狐狸、青蛙、小魚、烏龜、鴨子、猴子、企鵝
